#基于Flask+Markdown的简易博客系统

​	无聊的时间用Flask+Markdown写了个简易的博客系统（易博），练习练习差点忘掉的flask，易博仅支持Markdown文件。

运行环境：
​	python3
​	sqlite

python库：
​	Markdown==2.6.11
​	Flask==0.12.2

部署方式:
​	安装库:
​		pip3 install -r requirements.txt
​	修改config.py:

​		修改YiBo_Path为自己的目录.

​		修改SECRET_KEY.

​		修改登录USERNAME和PASSWORD.

使用方法:

​	登录后上传编辑好的markdown文件即可。
